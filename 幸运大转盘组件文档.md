# 幸运大转盘组件文档

## 1. 组件概述

本项目使用 `@lucky-canvas/taro` 库实现幸运大转盘抽奖功能，支持微信小程序等多端运行。

**官方文档**: https://100px.net/usage/taro.html
**详细文档**: https://100px.net/docs/wheel.html
**GitHub**: https://github.com/LuckDraw/lucky-canvas

---

## 2. 安装与引用

### 安装
```bash
npm install @lucky-canvas/taro@latest
# 或
yarn add @lucky-canvas/taro@latest
```

### 引用（Taro + Vue3）
```javascript
import { LuckyWheel } from '@lucky-canvas/taro/vue'
```

## 4. blocks - 背景配置

### 4.1 基础结构

```javascript
blocks: [
  { padding: '13px', background: '#617df2' }
]
```

### 4.2 完整字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `padding` | string \| number | 0 | 内边距（决定每层背景的直径） |
| `background` | string | 透明色 | 背景颜色（支持 hex/rgba） |
| `imgs` | Array | - | 背景图片列表 |

### 4.3 imgs 字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `src` | string | - | 图片路径 |
| `top` | string \| number | 0px | 图片距离顶部位置 |
| `width` | string \| number | 0px | 图片宽度 |
| `height` | string \| number | 0px | 图片高度 |
| `rotate` | boolean | false | 图片是否跟随旋转 |

### 4.4 背景绘制规则

背景从外向内绘制，每一层都会缩小相应尺寸。例如：
- 第一层 `padding: 13px` → 直径 = 父容器宽度
- 第二层 `padding: 10px` → 直径 = 父容器宽度 - 20px
- 第三层 `padding: 0px` → 直径 = 父容器宽度 - 40px（奖品区域）

## 5. prizes - 奖品配置

### 5.1 基础结构

```javascript
prizes: [
  {
    title: '一等奖',
    background: '#f8d384',
    fonts: [{ text: '一等奖', top: '35%' }],
    range: 30
  }
]
```

### 5.2 完整字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `range` | number | - | 中奖概率（用于按概率分配扇形） |
| `background` | string | 透明色 | 扇形背景颜色 |
| `fonts` | Array | - | 文字配置列表 |
| `imgs` | Array | - | 图片配置列表 |

### 5.3 fonts 字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `text` | string | - | 文字内容（可用\n换行） |
| `top` | string \| number | 0px | 文字距离顶部位置 |
| `fontColor` | string | #000 | 字体颜色 |
| `fontSize` | string \| number | 22px | 字体大小 |
| `fontStyle` | string | sans-serif | 字体样式 |
| `fontWeight` | string \| number | 400 | 字体粗细 |
| `lineHeight` | string \| number | 字体大小 | 字体行高 |
| `wordWrap` | boolean | true | 文字自动换行 |
| `lengthLimit` | string \| number | 90% | 换行宽度范围 |
| `lineClamp` | number | Infinity | 文本溢出隐藏行数 |

### 5.4 imgs 字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `src` | string | - | 图片路径 |
| `top` | string \| number | 0px | 图片距离顶部位置 |
| `width` | string \| number | 0px | 图片宽度 |
| `height` | string \| number | 0px | 图片高度 |

### 5.5 关键特性：range（按概率分配）

```javascript
// range 值表示该奖品的概率权重
// 转盘会根据所有奖品的 range 总和来分配扇形大小
prizes: [
  { title: '一等奖', range: 10, background: '#ffd700' },
  { title: '二等奖', range: 20, background: '#c0c0c0' },
  { title: '三等奖', range: 30, background: '#cd7f32' },
  { title: '未中奖', range: 40, background: '#e0e0e0' },
]
// 总概率 = 10 + 20 + 30 + 40 = 100
// 一等奖扇形 = 10/100 = 10%
// 二等奖扇形 = 20/100 = 20%
// ...
```

## 6. buttons - 抽奖按钮配置

### 6.1 基础结构

```javascript
buttons: [
  { radius: '50px', background: '#617df2' },
  { radius: '45px', background: '#afc8ff' },
  {
    radius: '40px',
    background: '#869cfa',
    pointer: true,
    fonts: [{ text: '开始\n抽奖', top: '-20px' }]
  }
]
```

### 6.2 完整字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `radius` | string \| number | 0px | **按钮半径（必填）** |
| `pointer` | boolean | false | 是否显示指针 |
| `background` | string | #000 | 按钮背景颜色 |
| `fonts` | Array | - | 文字配置（同 prizes.fonts） |
| `imgs` | Array | - | 图片配置（同 prizes.imgs） |

### 6.3 按钮绘制规则

- 按钮从外向内绘制，需注意半径大小避免覆盖
- `pointer: true` 表示这个按钮有指针指向转盘
- 最后一个按钮通常放文字/图片，避免被覆盖

## 7. defaultConfig - 默认配置

### 7.1 完整字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `gutter` | string \| number | 0px | 扇形之间的缝隙 |
| `stopRange` | number | 0 | 指针在扇形区域的停止范围（0~0.9） |
| `offsetDegree` | number | 0 | 转盘的偏移角度 |
| `speed` | number | 20 | 旋转速度峰值（建议10~30） |
| `accelerationTime` | number | 2500ms | 开始旋转时间 |
| `decelerationTime` | number | 2500ms | 缓慢停止时间 |

### 7.2 offsetDegree 计算公式

```javascript
// 让指针指向扇形中心或边缘的偏移角度
// 指针默认指向扇形中线

// 指向扇形边缘（常见配置）：
offsetDegree = (360 / 奖品数量) / 2

// 例如6个奖品：offsetDegree = 30
// 例如4个奖品：offsetDegree = 45
// 例如8个奖品：offsetDegree = 22.5
```

### 7.3 旋转三阶段

1. **加速阶段**: 从0加速到 `speed` 设置的峰值
2. **匀速阶段**: 保持峰值速度，直到调用 `stop()`
3. **减速阶段**: 从峰值逐渐减速到0

### 7.4 stopRange 说明

```javascript
// stopRange 控制指针在扇形区域内的停止位置
// 默认为0，表示指针停在扇形边缘
// 设置为0.5表示指针停在扇形中间
// 建议范围：0 ~ 0.9（设置为1可能会停在奖品边缘）
```

### 8.1 完整字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `background` | string | 透明色 | 奖品区域默认背景色 |
| `fontColor` | string | #000 | 字体颜色 |

- `prizes[].background` > `defaultStyle.background`

## 9. 回调函数 - callback

| 事件 | 类型 | 说明 |
|------|------|------|
| `@start` | (e) => void | 开始抽奖前（点击按钮时触发） |
| `@end` | (prize) => void | 结束抽奖后（转盘完全停止时触发） |

### 9.1 end 回调参数 prize 对象

```javascript
const endCallback = (prize) => {
  console.log(prize)
  // {
  //   index: 0,        // 中奖索引
  //   title: '一等奖', // 奖品名称
  //   range: 10       // 中奖概率
  // }
}
```

## 10. 方法 - methods

| 方法 | 参数 | 说明 |
|------|------|------|
| `init()` | 无 | 初始化组件 |
| `play()` | 无 | 开始抽奖旋转 |
| `stop(index)` | 中奖索引 | 缓慢停止到指定位置 |
| `stop(-1)` | -1 | 立即停止，不触发end回调 |

## 11. 完整使用示例

```vue
<template>
  <view>
    <LuckyWheel
      ref="myLucky"
      width="680rpx"
      height="680rpx"
      :blocks="blocks"
      :prizes="luckyPrizes"
      :buttons="buttons"
      :defaultConfig="defaultConfig"
      :defaultStyle="defaultStyle"
      @start="startCallback"
      @end="endCallback"
    />
  </view>
</template>

<script setup>
import { ref, computed } from 'vue'
import { LuckyWheel } from '@lucky-canvas/taro/vue'

const myLucky = ref(null)

// 背景配置
const blocks = [
  { padding: '15rpx', background: '#617df2' },
  { padding: '10rpx', background: '#afc8ff' },
  { padding: '0px', background: '#fff' }
]

// 奖品配置
const prizes = ref([
  { title: '一等奖', range: 10, background: '#ffd700' },
  { title: '二等奖', range: 20, background: '#c0c0c0' },
  { title: '三等奖', range: 30, background: '#cd7f32' },
  { title: '未中奖', range: 40, background: '#e0e0e0' }
])

// 转换为 LuckyWheel 格式
const luckyPrizes = computed(() => {
  return prizes.value.map(item => ({
    title: item.title,
    background: item.background,
    fonts: [{ text: item.title, top: '35%' }],
    range: item.range
  }))
})

// 按钮配置
const buttons = [
  { radius: '50px', background: '#617df2' },
  { radius: '45px', background: '#afc8ff' },
  {
    radius: '40px',
    background: '#869cfa',
    pointer: true,
    fonts: [{ text: '开始\n抽奖', top: '-20px' }]
  }
]

// 默认配置
const defaultConfig = {
  gutter: 5,
  stopRange: 0.8,
  offsetDegree: 45,
  speed: 20,
  accelerationTime: 2500,
  decelerationTime: 2500
}

// 默认样式
const defaultStyle = {
  fontColor: '#fff',
  fontSize: '24rpx',
  textAlign: 'center',
  wordWrap: true,
  lengthLimit: '80%'
}

// 开始抽奖
const startCallback = () => {
  myLucky.value.play()
  // 异步获取中奖结果
  setTimeout(() => {
    const index = Math.floor(Math.random() * prizes.value.length)
    myLucky.value.stop(index)
  }, 3000)
}

// 抽奖结束
const endCallback = (prize) => {
  console.log('中奖结果:', prize.title)
}
</script>
```

## 13. 常见问题

### Q: 转盘显示不正常？
**A**: 检查 `prizes` 数据是否正确加载，确保 `width` 和 `height` 单位正确（建议使用 rpx 或 px）。

### Q: 指针指向位置不对？
**A**: 调整 `defaultConfig.offsetDegree`，计算公式：`(360 / 奖品数量) / 2`

### Q: 扇形之间有缝隙？
**A**: 调整 `defaultConfig.gutter` 属性，设置为 0 可去除缝隙。

### Q: stop() 方法传入 -1 会怎样？
**A**: 转盘会立即停止，不会触发 `@end` 回调。

### Q: 如何实现按概率抽奖？
**A**: 使用 `prizes[].range` 属性，值越大该奖品的扇形区域越大。服务端决定中奖结果，前端根据结果调用 `stop(index)`。

### Q: 文字过长如何换行？
**A**: 使用 `fonts[].wordWrap: true`（默认开启），或使用 `\n` 手动换行。
